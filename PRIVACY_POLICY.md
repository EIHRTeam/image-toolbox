# Privacy Policy

**Last updated:** 2025-12-21

This Privacy Policy describes how **Image Toolbox** (the “Application”) handles information when you install and use it.

The Application is an open-source project maintained by the EIHRTeam and distributed via GitHub and the Microsoft Store.

---

## 1. Data Collection and Use

**The Application does not collect, process, store, transmit, or share any personal data or non-personal data.**

Specifically, the Application:

* Does **not** collect personal information such as names, email addresses, account identifiers, or contact details
* Does **not** collect image content, file data, metadata, or processing results
* Does **not** collect usage analytics, diagnostics, telemetry, logs, or crash reports
* Does **not** collect device identifiers, IP addresses, location data, or operating system information

All image processing and related operations are performed **entirely on the user’s local device**.

---

## 2. Network Access

The Application may access the internet **solely for the following limited purposes**:

* Checking for and downloading application updates
* Downloading required third-party dependencies

These network connections are used **only to retrieve software resources** and **do not involve the upload or transmission of any user data**.

The Application does not implement any background communication, tracking mechanisms, or remote data synchronization.

---

## 3. Third-Party Services

The Application **does not integrate or embed any third-party services**, including but not limited to:

* Advertising services
* Analytics or telemetry services
* Cloud storage or processing services
* User tracking or profiling services

Any third-party network activity initiated by the operating system, application runtime, package manager, or user-configured network environment is outside the control of the Application and is governed by the respective third party’s privacy policies.

---

## 4. Data Sharing

Because the Application does not collect any data, **no data is shared with any third party** under any circumstances.

---

## 5. Data Retention

The Application does not retain any personal or non-personal data, as no data is collected.

---

## 6. Security

Since the Application does not collect, store, or transmit user data, there is no risk of data breach, unauthorized access, or misuse of user information attributable to the Application.

---

## 7. Children’s Privacy

The Application is not directed at children and does not knowingly collect personal information from any individual, including children under applicable age thresholds.

---

## 8. Changes to This Privacy Policy

If future versions of the Application introduce changes that affect data handling or user privacy, this Privacy Policy will be updated accordingly and made publicly available prior to or at the time such changes take effect.

---

For questions or concerns regarding this Privacy Policy or the Application, please contact the project maintainers via the GitHub repository:

[https://github.com/EIHRTeam/image-toolbox](https://github.com/EIHRTeam/image-toolbox)
